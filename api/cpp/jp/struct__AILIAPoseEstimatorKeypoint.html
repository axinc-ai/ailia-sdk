◆ interpolated

int _AILIAPoseEstimatorKeypoint::interpolated

通常は0です。この点が未検出で、他の点から補間可能な場合、x,yの値を補間し、interpolated=1となります。

◆ score

float _AILIAPoseEstimatorKeypoint::score

この点の検出信頼度。値が0.0Fの場合、この点は未検出のため使用できません。
