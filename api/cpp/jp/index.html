概要

ailiaは、マルチプラットフォームに対応したDNN（Deep Neural Network、畳み込みニューラルネットワーク）の高速推論エンジンです。 学習済みモデルを読み込み、ネイティブアプリやUnity上でGPUを使用した高速な推論処理を行うことができます。

ailiaは依存関係が少なく、外部ツールのインストールが不要で、簡単にポーティングが可能です。 また、Windows、Mac、iOS、Android、Linuxで一貫したインターフェースを提供します。 加えて、前処理と後処理を行うユーティリティAPIも提供します。

主要なヘッダファイル

  • ailia.h 共通で使用される機能を提供します。
  • ailia_audio.h 音声を扱うモデルの前処理および後処理用の関数を提供します。
  • ailia_classifier.h 物体識別を簡単に行えるClassifier APIを提供します。
  • ailia_detector.h 物体検出を簡単に行えるユーティリティAPIを提供します。
  • ailia_feature_extractor.h 画像検索を簡単に行えるユーティリティAPIを提供します。
  • ailia_pose_estimator.h 骨格検出を簡単に行えるユーティリティAPIを提供します。
