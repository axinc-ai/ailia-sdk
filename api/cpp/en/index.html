Overview

ailia is a multi-platform, high-performance neural-network inference engine for deep neural networks (DNN). It can load a pretrained model and perform high speed inference processes with a GPU on native applications and Unity.

With fewer dependencies, ailia has no need for the installation of external tools, and can be ported easily. It also provides a consistent interface across Windows, Mac, iOS, Android, and Linux platforms. On top of all this, it offers utility APIs for pre-processing and post-processing.

Main headers

  • ailia.h provides common functions.
  • ailia_audio.h provides pre-processing and post-processing of speech models.
  • ailia_classifier.h provides the Classifier API for easy object classification.
  • ailia_detector.h provides a utility API for easy object detection.
  • ailia_feature_extractor.h provides a utility API for easy image searches.
  • ailia_pose_estimator.h provides a utility API for easy human pose estimation.
