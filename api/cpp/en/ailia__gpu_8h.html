GPU用カーネルのパラメーターを読み込みます 引数: net - ネットワークオブジェクトポインタ path - パラメーター情報の保存先パス名(MBSC or UTF16) 返値: 成功した場合はAILIA_STATUS_SUCCESS、そうでなければエラーコードを返す。 解説: ailiaOptimizeGpuParameteWriterFileで保存したパラメーターを読み込みます。 GPU以外の環境の場合はAILIA_STATUS_INVALID_STATEが返り、 ailiaCreateで指定した環境とパラメーターファイルを生成した環境が異なる場合は、 AILIA_STATUS_INVALID_PARAMINFOが返ります。

DEPRECATED int AILIA_API ailiaOptimizeGpuParameter ( struct AILIANetwork *  net)

GPU用カーネルのパラメーターを最適化します 引数: net - ネットワークオブジェクトポインタ 返値: 成功した場合はAILIA_STATUS_SUCCESS、そうでなければエラーコードを返す。 解説: ailiaCreate時に指定した環境IDのGPUに対してパラメーターの最適化を行います。 GPU以外の環境の場合はAILIA_STATUS_INVALID_STATEが返ります。

DEPRECATED int AILIA_API ailiaSaveGpuParameterFileA ( struct AILIANetwork *  net,
const char *  path 
)

GPU用カーネルのパラメーターを保存します 引数: net -ネットワークオブジェクトポインタ path - パラメーター情報の保存先パス名(MBSC or UTF16) 返値: 成功した場合はAILIA_STATUS_SUCCESS、そうでなければエラーコードを返す。 解説: ailiaOptimizeGpuParameterで最適化したパラメーターを保存します。 ailiaOptimizeGpuParameterを呼び出す前にこの関数を呼出した場合は初期値のパラメーターが保存されます。 また、 GPU以外の環境の場合はAILIA_STATUS_INVALID_STATEが返ります。
