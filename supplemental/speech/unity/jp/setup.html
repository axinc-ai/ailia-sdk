ailia_speech  1.3.0.0
Unity Packageのインポートとサンプルの実行

Unityのバージョン

Unity 2021.3.10f1以降が必要です。

Unity Pluginのインポート

unityフォルダのailia_speech.unity_packageをプロジェクトにインポートします。

依存ライブラリのPackage Managerからのインストール

ailia SDKをUnity Package Managerからインストールします。下記のURLを追加してください。

https://github.com/axinc-ai/ailia-sdk-unity.git
https://github.com/axinc-ai/ailia-audio-unity.git
https://github.com/axinc-ai/ailia-tokenizer-unity.git

Unity Pluginのフォルダ構成

ailia SpeechのUnity Packageは下記の構成を持ちます。

フォルダ 内容
ailia_speech 音声認識を行うailia Speechのインタフェースが含まれています。
Resources デモアプリの画像ファイルなどが含まれています。
StandaloneFileBrowser WindowsとmacOSでファイルダイアログを開くための拡張機能が含まれています。
StreamingAssets AIモデルと辞書ファイルが含まれています。
TextMeshPro テキストを描画するための拡張機能が含まれています。
Samples/Scripts/AiliaSpeechSample.cs メインとなるサンプルファイルです。
Samples/Scenes/AiliaSpeechScene.unity メインとなるサンプルシーンです。

ライセンスファイルの配置

評価版の利用にはライセンスファイルが必要です。 下記のフォルダにライセンスファイルを配置してください。

Windows

ailia.dllと同じフォルダ（サンプルの場合はAssets/Plugins/x86_64）にailia.licを配置します。

macOS

~/Library/SHALO/にailia.licを配置します。

Linux

~/.shalo/にailia.licを配置します。

サンプルの実行

AiliaSpeechScene.unityをUnityで開いて実行します。
