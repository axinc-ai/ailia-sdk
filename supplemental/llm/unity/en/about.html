About ailia LLM

Presentation of ailia LLM

ailia LLM is an inference library for LLMs that can be used from Unity and C++. It is possible to implement inference using a local LLM with a simple API.

Supported platforms

ailia LLM is available for Windows, macOS, Linux, iOS, and Android.

  • Windows 10 and later
  • macOS 13.3 and later
  • iOS 16.4 and later
  • Ubuntu 20.04 and later
  • Android NDK r23c
